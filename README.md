# jj_cyber_campus_management

# Run
* <b>share 폴더 내의 run.exe 파일 실행</b>
* Chrome(크롬)이 설치되어있어야 합니다.
* 네트워크(인터넷) 환경에 따라 시간이 소요될수도 있습니다.
